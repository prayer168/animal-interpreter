# 我是動物解說員

適合國小五年級自然科的分頁式互動教材。學生會觀察動物身體構造、比較環境條件、分析生存策略，最後使用「我觀察到……所以我推論……因為……」提出有證據的科學解說。

- 教材網站：https://prayer168.github.io/animal-interpreter/
- GitHub repository：https://github.com/prayer168/animal-interpreter
- 目前版本：v1.1.0

## 適用對象

- 年級：國小五年級
- 科目：自然科學
- 建議時間：40 分鐘
- 使用裝置：電腦、平板、手機

## 學習目標

1. 從動物身體構造的特徵，推論它能完成的功能。
2. 比較同類構造在不同環境中的差異，解釋適應關係。
3. 辨識保護色、擬態、警戒色與群體合作等生存策略。
4. 使用觀察證據提出解說，並指出推論的限制。

## 教材特色

- 8 個單一任務頁籤，不使用長篇一頁式首頁。
- 4 組構造圖解、3 組棲地條件切換與 4 組分階段生存策略動畫。
- 點選觀察、環境切換、拖曳／鍵盤配對、預測與即時回饋。
- 6 則生活觀察情境，每則都有獨立 SVG 圖解。
- 10 題理解、應用、推理與現象判讀闖關。
- 8 筆政府、博物館與研究機構自主學習資源。
- 支援學習進度保存、觸控操作、鍵盤導覽與降低動畫偏好。

## 頁籤內容

1. 學習任務
2. 構造 × 功能
3. 構造 × 環境
4. 生存法寶
5. 解說員實驗室
6. 生活觀察
7. 闖關挑戰
8. 自主學習

## 使用方式

直接開啟教材網站。課堂上可先由教師投影示範「觀察—推論—證據」句型，再讓學生個別或兩人共用載具完成互動與闖關。

## 本機啟動

```bash
npm install
npm run dev
```

正式建置：

```bash
npm run build
```

## 專案結構

- `index.html`：教材殼層、頁籤與分享資訊
- `css/`：主視覺、動畫與響應式規則
- `js/`：導覽、互動、測驗與進度保存
- `public/data/`：部署時公開載入的教材內容、題庫、資源與課綱對應
- `public/assets/images/`：Facebook／Open Graph 預覽圖
- `docs/`：教材藍圖、教案、參考資料與測試報告

## 使用技術

HTML5、CSS3、原生 JavaScript、SVG、JSON、Vite、GitHub Actions 與 GitHub Pages。

## 課綱與素養

教材以觀察、比較、推理、證據表達與生物環境關係為主，融入環境教育、生命教育與閱讀素養。正式學習表現與學習內容代碼待教師依教育部自然科學領域課程綱要及校本進度確認。
