# 教材藍圖

## 基本資料

- 中文教材名稱：我是動物解說員
- 英文專案名稱：Animal Interpreter
- GitHub repository：animal-interpreter（使用教材英文名稱）
- 適用年級：國小五年級
- 科目：自然科學
- 學習時間：40 分鐘
- 視覺風格：清爽、自然、教學寫實

## 學習目標

1. 從動物身體構造的特徵，推論它能完成的功能。
2. 比較同類構造在不同環境中的差異，解釋適應關係。
3. 辨識保護色、擬態、警戒色與群體合作等生存策略。
4. 使用「我觀察到……所以我推論……」提出有證據的解說。

## 先備知識與核心概念

學生已認識常見動物及基本棲地。核心概念是「構造支持功能；有利特徵與行為能提高在特定環境中生存、覓食與避敵的機會」。個體不會因為需要而立刻改變構造。

## 頁籤架構

1. 學習任務：目標、時間、任務與進度。
2. 活動1 構造×功能：鳥喙、腳、牙齒與功能的圖解與點選觀察。
3. 活動2 構造×環境：北極熊、駱駝、魚的環境條件切換與證據推論。
4. 活動3 生存法寶：保護色、擬態、警戒色與群體合作的階段動畫。
5. 解說員實驗室：拖曳配對、預測與即時回饋。
6. 生活觀察：6 則帶獨立 SVG 圖解的校園／生活情境。
7. 闖關挑戰：10 題理解、應用、推理與圖像判讀。
8. 自主學習：8 筆政府、博物館、研究與公共教育資源。

## 互動與動畫

- 點選構造標記，閱讀「構造—功能—證據」。
- 拖曳動物卡到合適棲地，提供鍵盤替代操作與重設。
- 切換寒冷／炎熱／水中環境，觀察 SVG 圖層與文字同步。
- 分階段生存策略動畫，具播放、暫停、重播與慢速控制。

## 生活應用與評量

生活應用涵蓋雨天鳥腳、寵物牙齒、校園昆蟲、魚市場、冬季動物與群體警戒。闖關共 10 題，誘答選項取自常見迷思，答題後顯示解釋與回看頁籤。

## 自主學習資源

優先採政府機關、博物館、動物園、公共媒體與研究機構網站；記錄查核日期與適用年級。

## 課綱對應

- 核心素養：運用觀察、比較與推理理解自然現象；表達有證據的解釋；關心生物與環境。
- 學習表現、學習內容代碼：待教師依正式課綱手冊確認。
- 議題融入：環境教育、生命教育、閱讀素養。
- 評量：以情境任務要求學生從可觀察特徵推論功能或生存優勢。
