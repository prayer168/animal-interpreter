# Changelog

## [1.1.0] - 2026-07-13

- 依可信照片與物種資料重繪綠頭鴨、大冠鷲、兔子與青蛙，補強真實辨識特徵。
- 重繪北極熊、單峰駱駝與鯖魚，修正動物體態、毛羽鱗片細節與構造標示。
- 將水中環境的說明標籤移到魚身之外，修正游動動畫造成標示錯位的問題。
- 重繪竹節蟲、枯葉蝶、瓢蟲警戒色及狐獴群體合作場景。
- 細緻化六則生活觀察插圖，完成桌機與手機版視覺及互動驗證。
- 新增動物外形重繪參考來源與授權說明。

## [1.0.1] - 2026-07-13

- Repository 改用教材英文名稱 `animal-interpreter`。
- 更新 GitHub Pages、Open Graph、Twitter Card、README 與專案設定網址。
- 修正正式建置未包含教材 JSON 的問題，將資料移至 `public/data/`。
- 保留原 `skill-1-2-3` repository 作為舊版部署紀錄。

## [1.0.0] - 2026-07-13

- 正式發布「我是動物解說員」互動教材。
- 完成構造與功能、環境適應、生存策略三個核心活動。
- 加入 4 組構造圖解、3 組環境切換與 4 組分階段 SVG 動畫。
- 加入滑鼠、觸控與鍵盤皆可操作的配對活動。
- 加入 6 則生活觀察、10 題闖關與 8 筆自主學習資源。
- 完成桌機、平板、手機響應式與降低動畫偏好支援。
- 加入 1200×630 Facebook／Open Graph 分享封面與完整 metadata。
- 補齊教師教案、教材藍圖、參考資料與品質檢核報告。
- 設定 GitHub Pages 自動發布流程。

## [0.1.0] - 2026-07-13

- 初始化互動教材專案。
- 完成教材藍圖與標準資料夾規劃。
